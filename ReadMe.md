# Автонастройка серверов


```mermaid
flowchart LR
  subgraph Playbook
    subgraph Roles
        subgraph Tasks
        end
    end
  end
```

- **Inventories** - "кого" настроить (список хостов/серверов)
- **Plugins** - сниппеты кода, расширяющие Ansible
- **Vars** - переменные для обозначения значений для playbook/role/task
